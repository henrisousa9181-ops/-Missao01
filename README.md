# Missão 01 - Prompt

## Prompt criado:
Explique o que é inteligência artificial de forma simples para calouros (iniciantes).

## Resultado:
Inteligência artificial (IA) é uma tecnologia que permite que computadores e máquinas aprendam, pensem e tomem decisões de forma parecida com os seres humanos.

Ela funciona analisando muitos dados e identificando padrões, o que ajuda a resolver problemas, responder perguntas e até prever situações. 

Hoje, a IA está presente em várias coisas do dia a dia, como assistentes virtuais, redes sociais, aplicativos de celular e até carros inteligentes.
